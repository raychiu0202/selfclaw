# Selfclaw Package

基于GLM-5的智能AI助手系统 - 发布包

## 安装

### NPM安装
```bash
npm install -g selfclaw
```

### pip安装
```bash
pip install selfclaw
```

## 快速开始

```bash
# 初始化配置
selfclaw init

# 启动服务
selfclaw start

# 查看服务状态
selfclaw status

# 停止服务
selfclaw stop
```

## 命令参考

- `selfclaw init` - 初始化配置
- `selfclaw start` - 启动服务
- `selfclaw stop` - 停止服务
- `selfclaw status` - 查看服务状态
- `selfclaw config` - 配置设置
- `selfclaw clean` - 清理数据
- `selfclaw logs` - 查看日志
