# Selfclaw - 任务规划文档

## 📋 项目概述

**项目名称：** Selfclaw - AI Agent 聊天工具（仿OpenClaw）
**开始时间：** 2024-04-15
**项目状态：** 核心功能已完成
**核心定位：** AI Agent（仿照OpenClaw）- 通过终端命令执行实现"对话+行动"的双重能力

## 🎯 总体目标

开发一个基于 GLM-5 模型的 AI Agent 系统，提供：
- **对话能力**：流式对话、多轮对话、上下文保持
- **行动能力**：通过终端命令执行，从"对话"扩展到"行动"
- **核心功能**：会话管理、Markdown 渲染、智能命令识别
- **动态显示**：仿OpenClaw的实时命令执行过程显示

**本质定位：** 为 AI 提供"手和脚"，使其能够执行具体的系统操作，而不仅仅是生成文本回复。

## 🎯 设计原则

为确保系统稳定性和可维护性，本项目严格遵循**极简设计原则**：

1. **功能最小化** - 仅保留核心必需功能，避免过度设计
2. **界面简洁性** - 最少视觉元素，清晰功能分区
3. **代码精简性** - 最小化复杂度，保持可读性
4. **性能优先** - 快速响应，低延迟
5. **安全性优先** - 白名单机制，严格权限控制
6. **Agent 能力优先** - 强调 AI 的行动能力，而非仅仅对话
7. **用户体验优先** - 仿OpenClaw的流畅交互体验
